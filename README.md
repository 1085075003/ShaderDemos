# Shader effects for cocos creator

### 九尾狐游戏工作室出品并提供长期维护

**使用说明**

    效果器shader包含两个文件，例如高斯模糊效果不包含GuassBlurEffect.ts和GuassBlurFrag.ts

    将对应的TS脚本拖到要施加效果的节点上即可以，方便快捷，自动施法，开箱即用 

    Scene/AmazingEffects/目录下能看到十几个酷炫的shader场景，用于参考学习
